# UX Renewal Design — Minimal Polish

**Date**: 2026-03-04
**Approach**: A. 미니멀 폴리시 — 현재 단일 페이지 구조 유지 + 핵심 UX 집중 개선

## 1. 입력 영역

- textarea → 단일 input + 붙여넣기 버튼 + 다운로드 버튼 (한 줄)
- 클립보드에 YouTube URL 감지 시 자동 제안
- 여러 URL 입력은 "접기/펼치기"로 숨김 처리
- 빈 상태일 때 간단한 사용 가이드

## 2. 비디오 정보 카드 & 포맷 선택

- 썸네일 크기 확대 (32px → 160px)
- 추천 포맷(최고 화질) 시각적 강조
- 오디오만 다운로드 옵션 추가
- 파일 크기 명확 표시
- 인라인 확장 (모달 아님)

## 3. 다운로드 목록 & 진행률

- 상태 아이콘: 대기/다운로드 중/완료/실패
- 진행 상세: 다운로드 속도(MB/s) + 남은 시간(ETA)
- 액션 버튼: 재시도, 삭제 (항목별)
- "완료 항목 정리" 일괄 버튼
- 완료 정보: 화질, 파일 크기, 완료 시각
- 에러 메시지를 사용자 친화적으로 변환

## 4. 전반적 폴리시

- **토스트 알림**: 완료/실패 시 우상단 (3초 자동 닫힘)
- **빈 상태**: 아이콘 + 안내 텍스트
- **로딩 스켈레톤**: URL 분석 중 shimmer 애니메이션
- **마이크로 인터랙션**: 버튼 hover/press scale, 카드 fade-in/slide-up, 완료 체크 애니메이션
- **에러 매핑**: 기술 에러 → 한국어 친화적 메시지

## 기술 스택 (변경 없음)

- React 19 + TypeScript + Tailwind CSS v4 + Vite
- WebSocket 진행률 (기존 유지)
- 외부 라이브러리 추가 없음 (순수 Tailwind 구현)

## 백엔드 변경

- 다운로드 속도/ETA를 WebSocket progress 메시지에 추가
- 다운로드 삭제 API 추가
- 다운로드 재시도 API 추가
- 완료 항목 일괄 삭제 API 추가
